    apply(self, image, opts)

    View Source

    Computes a foreground mask.

    • image.
    • fgmask.
    • learningRate. learnt. Negative parameter value makes the algorithm to use some automatically chosen learning rate. 0 means that the background model is not updated at all, 1 means that the background model is completely reinitialized from the last frame.
    • fgmask : UMat
    • learningRate : double

    Python prototype (for reference): 

    apply(image[, fgmask[, learningRate]]) -> fgmask
